# SimpleBackend

### prepare environment

1. install docker compose [refer](https://docs.docker.com/compose/install/)
2. check your node version > v14

### init this project

1. copy .env file
   
`
cp .env.example .env
`

2. fill empty variable on .env file

3. install modules

`
npm install
`

4. execute project

`
docker compose up -d
`

5. close project

`
docker compose down
`
